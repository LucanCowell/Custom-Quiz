# Custom-Quiz
This is a customiseable selection of quizzes that allow for users to create their own questions and study or a variety of other uses that the user may find.

In its base form it currently has personalised questions on the following subjects:
1. Chemistry
2. Business Studies
3. English Advanced
4. Math Advanced
5. Math Extension I
6. Studies of Religion X
7. Enterprise Computing

Each subject has 5 fully personalised questions. However with minor changes to the programming, there can be many more questions and is not limited to 5.
